Instructions on how to setup and start the software:
1. Install Python verison 3.6.8 on your pc or in a virtual environment.
2. Start Command Prompt/Terminal in the appropriate folder and install the required libraries in requirements.txt using the command: pip install -r requirements.txt
3. In Command Prompt/Terminal, go to Fake_News_ICT_App folder and type the following command: python manage.py runserver
4. Open browser and go to http://127.0.0.1:8000/ to start the web application.
